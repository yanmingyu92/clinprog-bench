📤 Submit Your Results

1

Install & Run

Install with pip install clinprog-bench, then run your agent against all 250 tasks using the runner contract in runners/contract.md.

2

Score Your Outputs

Use the built-in scorers to compute your per-category and aggregate scores. Ensure reproducibility by running 3 times and checking for bit-identical results.

3

Submit a Pull Request

Fork the repo, add your results to submissions/<agent-name>/, and open a PR. Include your score JSON, agent description, and any logs.
