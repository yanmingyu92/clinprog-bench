\section{Reproducibility}
\label{sec:reproducibility}

This appendix provides one-command instructions for reproducing the
baseline experiments reported in Table~\ref{tab:baselines}.

\subsection{Environment setup}

\begin{lstlisting}[language=bash, basicstyle=\ttfamily\small]
# Clone and install (Python >= 3.11)
git clone https://github.com/anonymous/clinprog-bench.git
cd clinprog-bench
pip install .                 # or: uv sync

# Verify: schema validation + unit tests
clinprog-bench validate tasks/
pytest -q                     # 79 tests, <1s
\end{lstlisting}

\subsection{Reference baseline (all 250 tasks)}

\begin{lstlisting}[language=bash, basicstyle=\ttfamily\small]
python -m clinprog_bench.runners.reference_baseline
# Output: results/reference_baseline_<timestamp>.json
# Expected macro zero-shot: 0.05
\end{lstlisting}

\subsection{LLM baselines (25-task stratified subset)}

Requires API keys set as environment variables:
\texttt{DEEPSEEK\_API\_KEY}, \texttt{ANTHROPIC\_API\_KEY},
\texttt{GLM\_API\_KEY}.

\begin{lstlisting}[language=bash, basicstyle=\ttfamily\small]
# Run all three LLM baselines (both conditions)
python scripts/run_llm_baseline.py \
    --models deepseek-chat claude-sonnet-4-5 glm-4-5 \
    --subset-seed 20260427 --subset-size 25 \
    --conditions zero-shot grounding \
    --output results/

# Results are written per-model to results/<model>_<condition>.json
\end{lstlisting}

\subsection{Continuous integration}

The GitHub Actions workflow
(\texttt{.github/workflows/eval.yml}) runs on every push and pull
request. It executes three jobs sequentially:
(1)~\textbf{schema-lint}: validates all 250 task JSONs, runs
\texttt{ruff}, \texttt{black}, and \texttt{mypy};
(2)~\textbf{unit}: runs the full \texttt{pytest} suite;
(3)~\textbf{baseline}: smoke-tests the reference baseline runner.
The CI badge in the repository README reflects the latest master status.

\subsection{Hardware and cost}

All baseline experiments were run on a consumer laptop
(Intel i7-12700H, 32\,GB RAM, no GPU) with API calls to external
model providers. Per-baseline API costs were approximately:
DeepSeek-Chat \$0.12 (25 tasks $\times$ 2 conditions),
Claude Sonnet~4.5 \$1.80, GLM-4.5 \$0.08.
Total wall-clock time: $\sim$40~minutes for the full 25-task
three-model matrix.
