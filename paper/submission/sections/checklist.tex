\section{NeurIPS D\&B Author Checklist}
\label{sec:checklist}

% Mirrors the NeurIPS Datasets & Benchmarks 2026 author checklist.
% Update each answer before submission; "TODO" markers below are placeholders.

\begin{itemize}
  \item \textbf{Dataset documentation.} Datasheet provided as
        \texttt{DATA\_CARD.md} in the repository root, following Gebru et al.
        (\S\ref{sec:dataset}).
  \item \textbf{Author responsibility statement.} The author confirms
        responsibility in case of violation of rights, see
        \S\ref{sec:ethics}.
  \item \textbf{Hosting and access.} Dataset hosted on
        GitHub (anonymized for review; URL provided as supplementary) and
        Zenodo (DOI withheld for double-blind review).
        Persistent dereferenceable identifier: a Zenodo DOI will be provided for camera-ready.
  \item \textbf{License.} Code: Apache-2.0. Data: CC-BY-4.0. Both committed
        as \texttt{LICENSE} and \texttt{LICENSE-DATA}.
  \item \textbf{Croissant metadata.} Machine-readable dataset card committed as
        \texttt{croissant.json} at the repository root, generated following the
        MLCommons Croissant specification (two record sets: \texttt{tasks}
        and \texttt{gold}).
  \item \textbf{Reproducibility.} CI workflow
        (\texttt{.github/workflows/eval.yml}) reproduces the reference
        baseline on every push. \texttt{CITATION.cff} pins the v0.1.0 SHA.
  \item \textbf{Computational resources.} See \S\ref{sec:ethics};
        per-baseline token budgets reported in
        Table~\ref{tab:baselines}.
  \item \textbf{Limitations.} \S\ref{sec:limitations}.
  \item \textbf{Broader impact.} \S\ref{sec:ethics}.
  \item \textbf{IRB / consent.} Not applicable (no human-subjects data).
  \item \textbf{Personal / sensitive information.} None
        (substrate is the public CDISC Pilot Project).
\end{itemize}
